Linux chattr 命令详解

描述

chattr命令用于更改Linux文件的属性，比如设置文件为不可删除、只读等类型。只有具有sudo权限的用户才能执行这项命令。chattr可看成是change attributes的缩写：ch+attr

lsattr可查看chattr命令执行后的文件属性
